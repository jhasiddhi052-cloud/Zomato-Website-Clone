## 📖 Project Overview

This project is inspired by the official Zomato website. It is designed to provide users with an attractive landing page where they can explore food categories, navigate through different sections, and enjoy a modern browsing experience.

The website is fully responsive and works across desktops, tablets, and mobile devices.
